Como conectar o Meetime a este dashboardGuia técnico
Via API REST do Meetime
# Endpoint base
GET https://api.meetime.com.br/v1/
# Auth header
Authorization: Bearer {TOKEN}
# Endpoints

1️⃣Meetime → Config → API → gere seu Bearer Token
2️⃣No Pipedream, crie workflow HTTP + steps Meetime
3️⃣Adicione as actions no MCP (igual ao Pipedrive)
4️⃣GitHub Actions busca via fetch() e commita data.json
5️⃣Dashboard lê o JSON automaticamente a cada push
